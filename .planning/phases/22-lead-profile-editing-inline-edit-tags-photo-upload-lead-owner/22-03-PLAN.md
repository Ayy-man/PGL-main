    **GET handler** — fetch tags for this prospect + optional tenant-wide suggestions:
    - Auth: `createClient()`, `getUser()`, extract `tenant_id`
    - Check URL searchParam `suggestions=true`: if true, also fetch all distinct tags for this tenant
    - Query: `supabase.from("prospect_tags").select("id, tag, created_at").eq("prospect_id", prospectId).eq("tenant_id", tenantId).order("created_at")`
    - For suggestions: `supabase.from("prospect_tags").select("tag").eq("tenant_id", tenantId)` then deduplicate with `[...new Set(data.map(r => r.tag))]`
    - Return: `{ tags: [...], suggestions?: [...] }`

    **POST handler** — add a tag:
    - Auth: same pattern
    - Parse body: `{ tag: string }`, validate with zod: `z.object({ tag: z.string().min(1).max(100) })`
    - Normalize: `tag = validated.tag.trim().toLowerCase()`
    - Insert: `supabase.from("prospect_tags").insert({ prospect_id: prospectId, tenant_id: tenantId, tag, created_by: user.id }).select("id, tag, created_at").single()`
    - Handle unique constraint violation (code `23505`): return 409 with `{ error: "Tag already exists" }`
    - Log: `logActivity({ actionType: "tag_added", metadata: { tag } }).catch(() => {})`
    - Return 201: `{ tag: data }`

    **DELETE handler** — remove a tag:
    - Auth: same pattern
    - Parse body: `{ tag: string }` with zod validation
    - Normalize: `tag = validated.tag.trim().toLowerCase()`
    - Delete: `supabase.from("prospect_tags").delete().eq("prospect_id", prospectId).eq("tenant_id", tenantId).eq("tag", tag)`
    - Log: `logActivity({ actionType: "tag_removed", metadata: { tag } }).catch(() => {})`
    - Return: `{ success: true }`

    CRITICAL: Use `createClient()` for all tag operations (RLS provides tenant isolation).

    9. Log: `logActivity({ actionType: "photo_uploaded", targetType: "prospect", targetId: prospectId }).catch(() => {})`
    10. Return: `{ url: publicUrl }`
