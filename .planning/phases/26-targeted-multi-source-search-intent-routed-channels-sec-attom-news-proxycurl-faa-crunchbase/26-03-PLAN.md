**File 1: `src/lib/search/channels/opencorporates-channel.ts`**

1. Import: `ChannelParams`, `ChannelOutput`, `ChannelResult`, `registerChannel` from `./index`, `getChannelCache`, `setChannelCache` from `../channel-cache`, `withCircuitBreaker` from `@/lib/circuit-breaker`, `openCorporatesRateLimiter` from `@/lib/rate-limit/limiters`.

2. Define inline response type:
```typescript
type OCCompany = {
  company: {
    name: string;
    company_number: string;
    jurisdiction_code: string;
    incorporation_date: string | null;
    dissolution_date: string | null;
    registered_address_in_full: string | null;
    current_status: string | null;
    opencorporates_url: string;
  };
};
```

3. Implement `searchOpenCorporatesInternal(params: ChannelParams): Promise<ChannelOutput>`:
   - Record `startMs = Date.now()`
   - Check cache first
   - Check `process.env.OPENCORPORATES_API_TOKEN`. If missing, return empty with error.
   - If `params.prospect.company` is null or empty, return empty (need company name to search)
   - Rate limit: `await openCorporatesRateLimiter.limit("opencorporates:global")`
   - Build URL: `https://api.opencorporates.com/v0.4/companies/search`
     - Query params: `q=${encodeURIComponent(params.prospect.company)}`, `api_token=${process.env.OPENCORPORATES_API_TOKEN}`, `jurisdiction_code=us` (scope to US)
   - Fetch with `{ headers: { "Accept": "application/json" } }`
   - Parse response: `{ results: { companies: OCCompany[] } }`
   - Map first 5 companies to `ChannelResult[]`:
     - `channelId: "opencorporates"`
     - `headline: company.name` (or `${company.name} (${company.jurisdiction_code})`)
     - `summary: "Company #${company.company_number} in ${company.jurisdiction_code}. Status: ${company.current_status || 'Unknown'}. Incorporated: ${company.incorporation_date || 'Unknown'}."`
     - `source_url: company.opencorporates_url`
     - `source_name: "OpenCorporates"`
     - `event_date: company.incorporation_date || null`
     - `category: "corporate"`
     - `relevance: "medium"`
     - `raw_snippet: JSON.stringify(company)`
   - Cache and return
   - try/catch: return empty on error

4. Export `searchOpenCorporates` via `withCircuitBreaker(searchOpenCorporatesInternal, { name: "opencorporates-channel", timeout: 10000 })`.

5. Register: `registerChannel("opencorporates", searchOpenCorporates)`.

**File 2: `src/lib/search/channels/crunchbase-channel.ts`**

1. Import: `ChannelParams`, `ChannelOutput`, `ChannelResult`, `registerChannel` from `./index`, `getChannelCache`, `setChannelCache` from `../channel-cache`, `withCircuitBreaker` from `@/lib/circuit-breaker`, `crunchbaseRateLimiter` from `@/lib/rate-limit/limiters`.

2. IMPORTANT: Crunchbase Basic (free) plan only returns limited fields. Limit query to: `short_description`, `company_type`, `founded_on`, `homepage_url`, `primary_role`. Do NOT request `funding_rounds`, `founders`, `investors` (403 on free tier per RESEARCH.md Pitfall 8).

3. Implement `searchCrunchbaseInternal(params: ChannelParams): Promise<ChannelOutput>`:
   - Record `startMs = Date.now()`
   - Check cache first
   - Check `process.env.CRUNCHBASE_API_KEY`. If missing, return empty with error.
   - If `params.prospect.company` is null or empty, return empty
   - Rate limit: `await crunchbaseRateLimiter.limit("crunchbase:global")`
   - POST to `https://api.crunchbase.com/api/v4/searches/organizations`
     - Query param: `user_key=${process.env.CRUNCHBASE_API_KEY}`
     - Headers: `{ "Content-Type": "application/json" }`
     - Body:
     ```json
     {
       "field_ids": ["short_description", "company_type", "founded_on", "homepage_url"],
       "query": [
         {
           "type": "predicate",
           "field_id": "identifier",
           "operator_id": "contains",
           "values": ["<company name>"]
         }
       ],
       "limit": 3
     }
     ```
   - Parse response: `{ entities: Array<{ uuid: string; properties: { identifier: { value: string; permalink: string }; short_description?: string; company_type?: string; founded_on?: { value: string }; homepage_url?: string } }> }`
   - Map entities to `ChannelResult[]`:
     - `channelId: "crunchbase"`
     - `headline: entity.properties.identifier.value`
     - `summary: entity.properties.short_description || "No description available". Company type: ${entity.properties.company_type || 'Unknown'}. Founded: ${entity.properties.founded_on?.value || 'Unknown'}.`
     - `source_url: "https://www.crunchbase.com/organization/${entity.properties.identifier.permalink}"`
     - `source_name: "Crunchbase"`
     - `event_date: entity.properties.founded_on?.value || null`
     - `category: "company_intel"`
     - `relevance: "medium"`
     - `raw_snippet: JSON.stringify(entity.properties)`
     - `confidence_note: "Crunchbase Basic tier — limited fields available"`
   - Cache and return
   - try/catch: return empty on error

4. Export `searchCrunchbase` via `withCircuitBreaker(searchCrunchbaseInternal, { name: "crunchbase-channel", timeout: 10000 })`.

5. Register: `registerChannel("crunchbase", searchCrunchbase)`.

Create `src/lib/search/channels/attom-channel.ts`:

ATTOM is an OPTIONAL premium channel ($95/mo). Most users will NOT have ATTOM_API_KEY configured. The channel must skip silently when the key is missing.

1. Import: `ChannelParams`, `ChannelOutput`, `ChannelResult`, `registerChannel` from `./index`, `getChannelCache`, `setChannelCache` from `../channel-cache`, `withCircuitBreaker` from `@/lib/circuit-breaker`.

2. No dedicated rate limiter for ATTOM (paid tier has generous limits). If needed later, add one.

3. Define inline response type:
```typescript
type ATTOMProperty = {
  identifier: { Id: number; fips: string; apn: string };
  lot: { lotSize1: number | null; lotSize2: number | null };
  address: { line1: string; line2: string; locality: string; countrySubd: string; postal1: string };
  assessment: { assessed: { assdTtlValue: number | null }; market: { mktTtlValue: number | null } };
  summary: { propclass: string | null; propsubtype: string | null; yearbuilt: number | null; propLandUse: string | null };
};
```

4. Implement `searchAttomInternal(params: ChannelParams): Promise<ChannelOutput>`:
   - Record `startMs = Date.now()`
   - Check cache first
   - Check `process.env.ATTOM_API_KEY`. If missing, return `{ channelId: "attom", results: [], cached: false, latencyMs: 0, error: "ATTOM_API_KEY not configured (premium channel)" }`.
   - Need a location to search. Extract from `params.prospect.location` or use `params.query` as fallback. If neither has usable address info, return empty with error "No address information available".
   - Build URL: `https://api.gateway.attomdata.com/propertyapi/v1.0.0/property/address`
     - Query params: `address1=${encodeURIComponent(addressLine)}`, `address2=${encodeURIComponent(cityState)}`
   - Fetch with headers: `{ "apikey": process.env.ATTOM_API_KEY!, "Accept": "application/json" }`
   - Parse response: `{ property: ATTOMProperty[] }`
   - Map first 3 properties to `ChannelResult[]`:
     - `channelId: "attom"`
     - `headline: "${property.address.line1}, ${property.address.locality}" or "Property at ${addressLine}"`
     - `summary: "Assessed value: $${property.assessment?.assessed?.assdTtlValue?.toLocaleString() || 'N/A'}. Market value: $${property.assessment?.market?.mktTtlValue?.toLocaleString() || 'N/A'}. Type: ${property.summary?.propclass || 'Unknown'}. Built: ${property.summary?.yearbuilt || 'Unknown'}."`
     - `source_url: "https://www.attomdata.com"` (ATTOM doesn't provide per-property URLs)
     - `source_name: "ATTOM Property"`
     - `event_date: null`
     - `category: "property"`
     - `relevance: "high"` (property data when queried is highly relevant)
     - `raw_snippet: JSON.stringify(property)`
   - Cache and return (7-day TTL via CHANNEL_TTLS)
   - try/catch: return empty on error

5. Export `searchAttom` via `withCircuitBreaker(searchAttomInternal, { name: "attom-channel", timeout: 10000 })`.

6. Register: `registerChannel("attom", searchAttom)`.
