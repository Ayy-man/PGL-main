```typescript
export async function POST(
  request: Request,
  { params }: { params: Promise<{ prospectId: string }> }
) {
```

2. Auth flow (copy from market-data/route.ts):
   - Create Supabase client via `createClient()`
   - `const { data: { user } } = await supabase.auth.getUser()`
   - If no user, return 401
   - Extract `tenantId` from `user.app_metadata.tenant_id`
   - If no tenantId and role is not super_admin, return 403

3. Parse request body: `const { query } = await request.json()` — expect `{ query: string }`
   - Validate: if no query or query.trim().length < 2, return 400 with error

4. Fetch prospect from DB:
```typescript
const { prospectId } = await params;
const { data: prospect, error } = await supabase
  .from("prospects")
  .select("id, full_name, title, company, location, publicly_traded_symbol, company_cik")
  .eq("id", prospectId)
  .single();
```
   - If error or no prospect, return 404

5. Call `executeResearch`:
```typescript
const result = await executeResearch({
  query: query.trim(),
  prospect: {
    id: prospect.id,
    full_name: prospect.full_name,
    company: prospect.company,
    title: prospect.title,
    publicly_traded_symbol: prospect.publicly_traded_symbol,
    company_cik: prospect.company_cik,
    location: prospect.location,
  },
  tenantId,
});
```

6. Return `NextResponse.json(result)` with status 200.

7. Wrap entire handler in try/catch. On error: return 500 with `{ error: "Research failed", details: String(error) }`.

Import `NextResponse` from `next/server`, `createClient` from appropriate Supabase helper, `executeResearch` from `@/lib/search/execute-research`.
