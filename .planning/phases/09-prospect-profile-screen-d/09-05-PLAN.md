## Goal

Apply responsive breakpoints to the full profile page so it works at 375px, 768px, 1024px, and 1440px per MASTER.md Section 13 responsive rules. The two-column layout collapses to single column on mobile. Tab bar scrolls horizontally on mobile. Profile header stacks vertically on mobile.

## must_haves

- Two-column layout collapses to single column below `lg` breakpoint (1024px): ActivityTimeline moves above or below tab content
- Profile header stacks avatar/name above action buttons on mobile
- Tab bar is horizontally scrollable on mobile (overflow-x-auto, no wrapping)
- Breadcrumbs and header padding reduce on mobile (px-14 -> px-4 on mobile)
- Info grid in overview tab goes from 3-col to 2-col on md, 1-col on mobile

## Tasks

<task id="09-05-T1">
**Add responsive breakpoints to `src/components/prospect/profile-view.tsx`**

1. **Two-column layout**: Change `flex gap-8 p-14` to:
   - `flex flex-col lg:flex-row gap-8 p-4 lg:p-14`
   - Left column: `w-full lg:w-[340px] lg:shrink-0` — on mobile, ActivityTimeline appears above tab content
   - Right column: `flex-1 min-w-0`

2. **Breadcrumbs wrapper**: Change `px-14 pt-6` to `px-4 lg:px-14 pt-6`

3. **Overview info grid**: Change `grid grid-cols-2 md:grid-cols-3 gap-4` to `grid grid-cols-1 sm:grid-cols-2 md:grid-cols-3 gap-4`

4. **ActivityTimeline on mobile**: On mobile, limit to 5 most recent events with "View all activity" link that switches to the Activity tab. On desktop (lg+), show all events.
</task>

<task id="09-05-T2">
**Add responsive breakpoints to `src/components/prospect/profile-header.tsx`**

1. **Container padding**: Change `py-7 px-14` to `py-5 px-4 lg:py-7 lg:px-14`

2. **Layout**: Change the flex layout to stack on mobile:
   - Main flex: `flex flex-col lg:flex-row lg:items-start justify-between gap-4 lg:gap-6`
   - Avatar row: always horizontal (avatar + name/title block)
   - Action buttons: `flex flex-wrap items-center gap-2` — they wrap naturally on mobile

3. **Name size**: `text-[24px] lg:text-[28px]` for slightly smaller on mobile

4. **"Draft Outreach" button**: Hide on mobile (`hidden lg:flex`) — action buttons area is already tight on mobile
</task>

<task id="09-05-T3">
**Add responsive breakpoints to `src/components/prospect/profile-tabs.tsx`**

1. **Container**: Change `px-14` to `px-4 lg:px-14`

2. **Horizontal scroll on mobile**: Add `overflow-x-auto` and `scrollbar-hide` (or `-webkit-overflow-scrolling: touch` with `overflow-x: auto`) to the tab container. Tabs should not wrap — they scroll horizontally.

3. **Tab whitespace**: Add `whitespace-nowrap` to each tab button to prevent text wrapping.

4. **Sticky top offset**: The tab bar is `sticky z-10`. Ensure `top` value accounts for TopBar height (56px). Add `top-14` (56px) or use `style={{ top: "56px" }}` to position below the TopBar.
</task>
