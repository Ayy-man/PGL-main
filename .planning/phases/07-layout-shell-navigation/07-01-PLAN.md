Note: `exact` field is new â€” Dashboard must use exact pathname match (not `startsWith`) to prevent it being active on every page.

3. Update the `isActive` logic inside the `.map()` to respect the `exact` field:
```typescript
const isActive = item.exact
  ? pathname === fullHref
  : pathname === fullHref || pathname.startsWith(`${fullHref}/`);
```

For Dashboard, `fullHref` will be `/${orgId}/` but `pathname` will also be `/${orgId}` (no trailing slash). Handle both:
```typescript
const isActive = item.exact
  ? (pathname === fullHref || pathname === fullHref.replace(/\/$/, ''))

```

Actually, since href is "/" and fullHref will be `/${orgId}/`, we need:
```typescript
const fullHref = item.href === "/" ? `/${orgId}` : `/${orgId}${item.href}`;
const isActive = item.exact
  ? pathname === fullHref
  : pathname === fullHref || pathname.startsWith(`${fullHref}/`);
```

This ensures Dashboard links to `/${orgId}` and is only active on that exact route.
