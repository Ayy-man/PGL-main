### Task 4: Create AutomationHealthCard Component

**File:** `src/components/admin/automation-health-card.tsx`

Card for each automation function. Shows health badge, type, metrics, last/next run.

**Health badge logic:**
- `successRate >= 90` → "Healthy" (green)
- `successRate >= 70` → "Degraded" (yellow/amber)
- `successRate < 70` → "Down" (red)
- `runs24h === 0 && lastRun === null` → "Never Run" (muted)

**Card layout:**

```tsx
<div className="surface-card rounded-[14px] p-4 md:p-6 space-y-3">
  {/* Header: name + health badge */}
  <div className="flex items-center justify-between">
    <h3 className="font-serif text-base font-semibold">{fn.name}</h3>
    <span className={cn("text-xs font-medium px-2 py-0.5 rounded-full", healthColor)}>
      {healthLabel}
    </span>
  </div>

  {/* Description */}
  <p className="text-xs" style={{ color: "var(--text-secondary-ds)" }}>{fn.description}</p>

  {/* Metrics grid */}
  <div className="space-y-2 text-sm">
    <div className="flex justify-between">
      <span style={{ color: "var(--text-secondary-ds)" }}>Type</span>
      <span className="font-mono text-xs">{fn.type === "cron" ? fn.trigger : "Event-driven"}</span>
    </div>
    <div className="flex justify-between">
      <span style={{ color: "var(--text-secondary-ds)" }}>Runs (24h)</span>
      <span>{fn.runs24h}</span>
    </div>
    <div className="flex justify-between items-center">
      <span style={{ color: "var(--text-secondary-ds)" }}>Success rate</span>
      <span>{fn.successRate}%</span>
    </div>
    {/* Success rate bar */}
    <div className="h-1.5 rounded-full" style={{ background: "var(--border-subtle)" }}>
      <div className="h-full rounded-full transition-all" style={{
        width: `${fn.successRate}%`,
        background: fn.successRate >= 90 ? "oklch(0.72 0.17 142)" : fn.successRate >= 70 ? "oklch(0.80 0.15 85)" : "oklch(0.62 0.19 22)",
      }} />
    </div>
  </div>

  {/* Last run + Next run */}
  <div className="pt-2 border-t space-y-1 text-sm" style={{ borderColor: "var(--border-subtle)" }}>
    <div className="flex justify-between">
      <span style={{ color: "var(--text-secondary-ds)" }}>Last run</span>
      <span>
        <StatusBadge status={fn.lastRun?.status} />
        {" "}{fn.lastRun ? relativeTime(fn.lastRun.finishedAt || fn.lastRun.startedAt) : "Never"}
      </span>
    </div>
    {fn.nextRun && (
      <div className="flex justify-between">
        <span style={{ color: "var(--text-secondary-ds)" }}>Next run</span>
        <span>{relativeTime(fn.nextRun)}</span>
      </div>
    )}
  </div>

  {/* Source health (enrich-prospect only) */}
  {fn.sourceHealth && (
    <div className="pt-2 border-t" style={{ borderColor: "var(--border-subtle)" }}>
      <div className="flex gap-3">
        {Object.entries(fn.sourceHealth).map(([source, { success, failed }]) => (
          <SourceDot key={source} source={source} success={success} failed={failed} />
        ))}
      </div>
    </div>
  )}
</div>
```

**StatusBadge:** Inline badge — "Completed" green, "Failed" red, "In Progress" amber, with small dot.

**SourceDot:** Small colored circle (green/red/gray) with tooltip showing source name + count.

**relativeTime:** Helper function — "10 minutes ago", "about 2 hours ago", "in about 6 hours" (same pattern as existing admin components).
