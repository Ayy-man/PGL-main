# Plan 19-04: Recent Runs Table + AutomationDetailDrawer Sidebar

**Phase:** 19 — Admin Automations Dashboard
**Goal:** Build the recent runs table and clickable detail sidebar with source breakdown and actions

---

## Context

Bottom section of the Automations page: a table of recent runs across all automations. Clicking a row opens a right-side Sheet drawer with full run details, source-level status, and a re-enrich action.

Follows TenantDetailDrawer pattern (Sheet, side="right", p-6 space-y-5).

---

## Tasks

### Task 1: Create AutomationRunsTable Component

**File:** `src/components/admin/automation-runs-table.tsx`

Client component that fetches `/api/admin/automations/runs?limit=20` on mount.

**Desktop table layout:**

| Automation | Target | Status | Duration | Started |
|---|---|---|---|---|
| Enrich Prospect | John Smith | Completed | 4.2s | 10 min ago |
| Daily Metrics | 2026-03-11 | Completed | 1.7s | 18 hrs ago |

**Columns:**
- **Automation**: function name with icon (Sparkles for enrich, Clock for cron)
- **Target**: prospect name (linked) or date string
- **Status**: colored badge (Completed/Failed/In Progress)
- **Duration**: formatted from durationMs (e.g., "4.2s", "1.7s", "—" if null)
- **Started**: relative time

**Table styling** follows existing admin table pattern:
- `text-[11px] font-semibold uppercase tracking-wider` for headers
- `hover:bg-[rgba(255,255,255,0.02)]` on rows
- `cursor-pointer` on rows — clicks call `onRunClick(run.id, run.functionId)`

**Mobile card layout** (hidden on md+, shown on mobile):
```tsx
<div className="md:hidden divide-y" style={{ borderColor: "var(--border-subtle)" }}>
  {runs.map(run => (
    <div key={run.id} className="p-4 space-y-2" onClick={() => onRunClick(run.id, run.functionId)}>
      <div className="flex items-center justify-between">
        <span className="font-medium text-sm">{run.functionName}</span>
        <StatusBadge status={run.status} />
      </div>
      <div className="flex items-center justify-between text-xs" style={{ color: "var(--text-secondary-ds)" }}>
        <span>{run.label}</span>
        <span>{run.durationMs ? formatDuration(run.durationMs) : "—"}</span>
      </div>
      <div className="text-xs" style={{ color: "var(--text-ghost)" }}>
        {relativeTime(run.startedAt)}
      </div>
    </div>
  ))}
</div>
```

**Section header:**
```tsx
<div className="space-y-1 mb-4">
  <h2 className="text-sm font-semibold uppercase tracking-wider"
      style={{ color: "var(--text-ghost)" }}>
    Recent Runs
  </h2>
  <p className="text-xs" style={{ color: "var(--text-secondary-ds)" }}>
    Last 20 task executions across all automations.
  </p>
</div>
```

**Loading state:** Skeleton rows matching table dimensions.

### Task 5: DrawerContent — Timing Grid

2x3 grid with timing details:

```tsx
<div className="mx-4 md:mx-6 mt-4 pt-4 border-t" style={{ borderColor: "var(--border-subtle)" }}>
  <div className="grid grid-cols-2 gap-4">
    <TimingField label="Created" value={formatTimestamp(data.startedAt)} />
    <TimingField label="Started" value={formatTimestamp(data.startedAt)} />
    <TimingField label="Finished" value={data.finishedAt ? formatTimestamp(data.finishedAt) : "—"} />
    <TimingField label="Duration" value={data.durationMs ? formatDuration(data.durationMs) : "—"} />
    <TimingField label="Attempts" value="1" />
    {data.inngestRunData?.output && (
      <TimingField label="Steps" value={String(data.inngestRunData.output?.sources ? Object.keys(data.inngestRunData.output.sources).length : "—")} />
    )}
  </div>
</div>
```

**TimingField sub-component:**
```tsx
function TimingField({ label, value }: { label: string; value: string }) {
  return (
    <div>
      <p className="text-xs" style={{ color: "var(--text-ghost)" }}>{label}</p>
      <p className="text-sm font-medium mt-0.5">{value}</p>
    </div>
  );
}
```

### Task 6: DrawerContent — Source Breakdown (enrichment only)

Vertical list of each enrichment source with status icon, timing, and error:

```tsx
{data.sourceBreakdown && (

      Source Breakdown

      <div key={source.name} className="flex items-start gap-3 p-3 rounded-lg"
           style={{ background: "rgba(255,255,255,0.02)" }}>
        <SourceStatusIcon status={source.status} />
        <div className="flex-1 min-w-0">
          <div className="flex items-center justify-between">
            <span className="text-sm font-medium capitalize">{source.name}</span>
            <span className="text-xs" style={{ color: "var(--text-ghost)" }}>
              {source.status}
            </span>
          </div>
          {source.error && (
            <p className="text-xs mt-1 truncate" style={{ color: "oklch(0.62 0.19 22)" }}>
              {source.error}
            </p>
          )}
          {source.at && (
            <p className="text-xs mt-0.5" style={{ color: "var(--text-ghost)" }}>

      </div>

  </div>
)}
```

**SourceStatusIcon:** CheckCircle (green) for complete, XCircle (red) for failed, MinusCircle (gray) for skipped, Loader (amber) for pending/in_progress.

### Task 8: Wire Drawer into Page

**File:** `src/app/admin/automations/page.tsx`

Add state for selected run and drawer open:

```typescript
const [selectedRunId, setSelectedRunId] = useState<string | null>(null);
const [selectedRunType, setSelectedRunType] = useState<string | null>(null);
const [drawerOpen, setDrawerOpen] = useState(false);

const handleRunClick = (runId: string, functionId: string) => {
  setSelectedRunId(runId);
  setSelectedRunType(functionId);
  setDrawerOpen(true);
};
```

Pass `handleRunClick` to `AutomationRunsTable` and drawer state to `AutomationDetailDrawer`.
