Required sections:

1. Prep
   - Create or reset a test tenant in Supabase Dashboard: set `auth.users.app_metadata` of a test user to remove `onboarding_state` entirely.

   - On first dashboard render, a popover appears anchored to the "New Search" / discover card.
   - Step counter reads "Step 1 of 6".
   - Clicking Next advances to the search page and anchors on the NL search bar.
   - Walking through all 6 steps ends with a "Done" button on the Export step.
   - Clicking Done: popover closes; refresh the page — tour does NOT re-appear.
   - Reset the user's `onboarding_state.tour_completed` via Supabase. Log back in — tour fires again.
   - Escape key dismisses the tour; refresh — tour does NOT re-fire.
   - With tour active on the dashboard, navigate to a role-gated page (e.g., /team as an agent). The tour either gracefully advances or shows the fallback centered card — not a blank popover hanging over nothing.

3. Help menu
   - CircleHelp icon visible in top-bar on every tenant page.
   - Click opens dropdown with 3 items in order: Watch intro video, Replay product tour, Report an issue.
   - Watch intro video: without `NEXT_PUBLIC_PGL_INTRO_VIDEO_URL` set, dialog shows "Video coming soon" fallback. With URL set, iframe/video renders.
   - Replay product tour: page refreshes, tour fires from Step 1.
   - Report an issue: opens the existing ReportIssueDialog with a screenshot pre-captured.

4. Admin checklist (as tenant_admin)
   - On dashboard, checklist card visible with progress bar at 0 of 4.
   - Invite a team member — reload dashboard — item 1 checks off, progress 1 of 4.
   - Upload a logo via Settings > Organization — reload — item 2 checks off, 2 of 4.
   - Change theme from gold to another — save — reload — item 3 checks off, 3 of 4.
   - Create a persona — reload — item 4 checks off. Checklist disappears entirely.

5. Non-admin role (as agent)
   - No admin checklist visible on dashboard (role gate verified).
   - Tour still fires on first login.
   - Help menu still visible.

6. Empty states
   - Dashboard (zero personas): EmptyState shows "Find your first prospects" + CTA button linking to /personas.
   - Lists (zero lists): EmptyState with "Create your first list" CTA.
   - Personas (zero personas): EmptyState with "Create your first saved search" CTA.
   - Activity (zero rows): EmptyState with "Start a search" CTA linking to /search.

7. Tooltips
   - Hover Advanced Filters toggle — tooltip appears with the filter explanation.
   - Hover each enrichment status dot on a prospect profile — each shows its own source-specific tooltip.
   - Hover Enrich Selection button in bulk actions bar — tooltip explains credit cost.
   - Hover a wealth tier badge on any prospect card — tooltip explains tier.

8. Regression
   - Existing `/onboarding/confirm-tenant` and `/onboarding/set-password` routes still work for new invited users.
   - Existing top-bar avatar link to /settings still works.
   - Existing ReportIssueButton on prospect pages still works.
