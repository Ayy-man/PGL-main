- Each group: date label header (text-[11px] uppercase tracking-wider text-muted-foreground mb-2) followed by events

DATA (category='data'):
- Blue dot (10px, var(--info, #3b82f6))
- Blue left accent bar
- System-generated title (from event.title)

TEAM (category='team'):
- Gray hollow dot (10px, ring-1 border only, no fill, rgba(255,255,255,0.25))

CUSTOM (category='custom'):
- Purple dot (10px, var(--purple, #a855f7))
