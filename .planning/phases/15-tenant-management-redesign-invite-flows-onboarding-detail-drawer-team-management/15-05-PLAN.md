# Plan 15-05: Tenant Detail Slide-Over Drawer

## Goal
Build the tenant detail slide-over drawer that opens when clicking a tenant row. Contains header, seat utilization, health score, usage stats, top personas, growth trend, and quota cards.

## Tasks

### Task 1: Make tenant rows clickable
**File:** `src/app/admin/tenants/page.tsx`

Convert to client component (or extract client wrapper):
- Add `selectedTenantId` state
- On row click, set `selectedTenantId` to tenant.id
- Render `TenantDetailDrawer` with `open={!!selectedTenantId}` and `onClose` to clear state
- Row gets `cursor-pointer` and existing `admin-row-hover` class

### Task 2: Create TenantDetailDrawer component
**File:** `src/components/admin/tenant-detail-drawer.tsx` (new)

Uses shadcn `Sheet` (side="right"):
- Width: `w-[580px]` (wider than default sm)
- Props: `tenantId: string`, `open: boolean`, `onOpenChange: (open: boolean) => void`
- On open, fetches all data in parallel from 5 endpoints:
  - `/api/admin/tenants/{id}` (details + users)
  - `/api/admin/tenants/{id}/usage` (stats)
  - `/api/admin/tenants/{id}/health` (score)
  - `/api/admin/tenants/{id}/personas` (top 3)
  - `/api/admin/tenants/{id}/activity?tab=all&limit=5` (recent activity)
- Shows skeleton loading state while fetching
- Scrollable content area with `overflow-y-auto`
