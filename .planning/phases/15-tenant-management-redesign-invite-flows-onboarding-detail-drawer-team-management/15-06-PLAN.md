# Plan 15-06: Tabbed Activity Card with 3-Level Drill-Down Modal

## Goal
Build the tabbed activity card in the tenant detail drawer with All/Admin/User tabs showing 5 recent entries each, plus an expand button that opens a centered modal with the full activity log.

## Tasks

### Task 1: Create TenantActivityCard component
**File:** `src/components/admin/tenant-activity-card.tsx` (new)

Card with `surface-admin-card` styling, rounded-[14px]:
- Title: "Activity" with an expand button (Maximize2 icon) in top-right
- shadcn `Tabs` component with 3 tabs: All | Admin Actions | User Activity
- Each tab fetches from `/api/admin/tenants/{id}/activity?tab={tab}&limit=5`
- Cache results per tab so switching doesn't re-fetch
